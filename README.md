# silverbirchh.github.io-hepoils
Hep Oils public website


## Development
Make sure that you have Jekyll installed, you can follow the quick start instructions [here](https://jekyllrb.com/). 
Then run...
```
~ cd silverbirchh.github.io-hepoils
~ bundle exec jekyll serve
```

Now browse to http://localhost:4000

## Deployment
Any merges to main will automatically be deployed by the CI/CD on Vercel.
